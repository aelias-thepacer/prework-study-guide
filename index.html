HTML

  • HTML relies on bracketed elements for structure.
  • Remember to look for these brackets when debugging HTML!
  • HTML also uses IDs within those elements in order to communicate data to JavaScript and CSS.
  • Different elements do different things. For example, the head element contains metadata.
  • Other elements include the body, which contains the other elements you will use to create your webpage,
  • As well as elements like Headers and bulleted lists.

CSS

  • CSS determines the visual aspects of a website.
  • Use it to make your creations look better.
  • In CSS, you set up one or more distinct rules determining how certain types or tags of elments will behave.

Git

  • Git status: what branch you are on
  • Git clone: copies a repository
  • Git checkout -b branch-name: creates and swaps to a new branch

JavaScript

  • Variables are containers assigned to names that let us store data.
  • Functions are self-contained chunks of code that are executed when called.
  • Code must be in a very specific order in order to work properly.
